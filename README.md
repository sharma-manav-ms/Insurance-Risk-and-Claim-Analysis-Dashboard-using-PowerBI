# 🚀 Insurance Risk & Claims Analysis Dashboard (Power BI)

## 📌 Project Overview

Designed an interactive Power BI dashboard to uncover insurance claim trends, analyze customer risk patterns, and generate actionable business insights.

---

## 🎯 Objectives

* Analyze insurance claims and customer behavior
* Identify high-risk segments
* Build an interactive dashboard for decision-making

---

## 🛠️ Tech Stack

* Power BI
* DAX (Data Analysis Expressions)
* Power Query
* Excel Dataset

---

## 📂 Dataset Features

* Customer Demographics (Age, Gender, Marital Status)
* Vehicle Details (Car Type, Usage)
* Insurance Coverage Zones
* Claim Amount & Frequency
* Income & Driving Factors

---

### 🔹Insights Generated

* Identified high-risk customer groups
* Analyzed claim distribution across demographics
* Found patterns in claim frequency and amount

---

## 📊 Key Features

* Interactive dashboard with filters
* Dynamic KPI tracking
* Risk segmentation analysis
* Clean and professional UI design
